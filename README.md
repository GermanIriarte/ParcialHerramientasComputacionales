# ParcialHerramientasComputacionales
## Descripción del problema 
El archivo **dos** es una solución en código al problema de la construcción de una plataforma que registra la compra de productos por parte de profesores y estudiantes de una institucion, supliendo el mecanismo de compra manual en una cafetería.  
## Modelo computacional
El modelo computacional que resuleve el problema es el de computación algorítmica, pues se soluciona a través de la implementación de código en la plataforma python. 
## Entradas del programa
* El código del producto a comprar.
* La cantidad de unidades que desea comprar.
* El precio respectivo. 
* La confirmación de terminación de pedido.
## Salidas 
* El tipo de usuario que compró (profesor o estudiante).
* La cédula.
* El valor a cancelar por los productos. 
* Los productos seleccionados. 
## *Cálculos*
el programa recoge todas las entradas mencionadas y a través del uso de un cilo va acumulando en una variable la suma del precio de cada pruducto, además de ir guardando en una lista vacía los códigos correspondientes a los productos elegidos. Cuando ya se ha decidido finalizar con el registro de compra, se imprime según se haya indicado, el tipo de usuario (profesor o estudiante) y respectivamente se calcula el precio final multiplicando por el descuento que les corresponde.
